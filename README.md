# Тестовая форма отзыва «Ваш Контроль»

Статический прототип формы отзыва для `vashkontrol.ru`. Интерфейс ведёт пользователя по короткому сценарию: выбрать локацию, найти организацию или услугу, выбрать точную пару «организация + услуга», поставить пять обязательных оценок и отправить отзыв.

## Как открыть

Откройте `index.html` в браузере или запустите локальный статический сервер:

```bash
uv run python -m http.server 8000
```

## Что работает

- Локальный поиск услуги и организации.
- Режимы поиска: «Не знаю», «Получал в МФЦ», «Получал в ведомстве».
- Отдельное редактируемое поле региона с нативными подсказками по регионам России и крупным городам.
- Автоподстановка региона по IP через публичный HTTPS-адрес сервиса, с запасным вариантом по часовому поясу браузера.
- Основной сценарий «организация → услуга», вкладка «Услуги» остаётся доступной.
- Выбор точной пары «услуга + организация».
- Пять обязательных параметров оценки по шкале 5 звёзд.
- Необязательные комментарий, фото и ссылка на видео.
- Видимый юридический блок «Хочу получить официальный ответ».
- Сохранение черновика в `localStorage`.
- Локальное подтверждение отправки и запись данных отзыва в `console.log`.

## Логика поиска

Справочники услуг и организаций находятся в `script.js`. Серверного каталога в прототипе пока нет.

1. Поле региона заполняется статическим списком локаций.
2. Основной поиск удаляет из запроса известные регионы и города России.
3. Совпадения ищутся по названиям услуг, синонимам, названиям организаций, ведомствам и псевдонимам.
4. Выбранный регион повышает релевантность подходящих организаций, но не добавляет обязательный шаг.
5. Режим поиска повышает релевантность МФЦ или ведомств.
6. Онлайн-тип получения немного выше поднимает федеральный портал и ведомства.
7. Если пользователь не нашёл нужный вариант, можно отправить локальное сообщение о проблеме справочника.

Отзыв нельзя отправить, пока не выбрана конкретная услуга и конкретная организация.
